# IconToolForiOS13.1.3
适配了ios 13.2.3  
iOS 13.1.3 logo上滑获得该应用的bundleID  
可修改应用名称，但是不完美，有些特殊的app需要滑动界面来刷新一次  
可以自定义角标  
可以在Filza中打开当前程序Bundle路径  
可以在Filza中打开当前程序沙盒路径 
可以自定义桌面图标每行的个数（需要滑动界面刷新）
